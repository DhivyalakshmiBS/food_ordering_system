🍽️ Online Food Ordering System in C

📌 Project Overview

This is a console-based Online Food Ordering System developed in the C programming language. The project simulates the basic workflow of an online restaurant ordering application where users can browse food categories, select items, place multiple orders, choose delivery and payment options, and receive a final bill.

The main objective of this project is to strengthen programming fundamentals by applying concepts such as loops, conditional statements, arrays, switch-case, and user interaction in a real-world scenario.

---

✨ Features

- Customer details (Name & Phone Number)
- Food category selection
- Multiple food varieties under each category
- Item selection
- Quantity selection
- Multiple item ordering using a "do-while" loop
- Automatic bill calculation
- GST calculation (18%)
- Delivery type selection
- Payment method selection
- Final invoice generation

---

🧾 Billing Process

1. Enter customer details.
2. Select a food category.
3. Choose a food item.
4. Enter quantity.
5. Add more items if required.
6. Calculate total bill.
7. Apply GST.
8. Select delivery type.
9. Select payment method.
10. Display the final invoice.
